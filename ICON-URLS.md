## Usage Notes:

1. **Size Parameters**: The `w=` parameter controls the width. Adjust as needed.
2. **Quality**: Add `&q=80` for 80% quality (good balance of size/quality)
3. **Format**: Unsplash auto-serves WebP to supported browsers
4. **Caching**: URLs are cached by CDN for fast loading

## SVG Icons (Recommended for Production)

For production, consider using:
1. **Font Awesome** (free icons): https://fontawesome.com
2. **Lucide Icons** (clean, modern): https://lucide.dev
3. **Heroicons** (Tailwind's icons): https://heroicons.com
4. **Custom SVGs** from Figma or design tools

Example with Font Awesome:
```html
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.5.1/css/all.min.css">
<i class="fa-solid fa-house"></i>
```
